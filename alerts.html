1. GitHub 연결 미연결

키워드를 저장소(data/alerts.json)에 반영하려면 GitHub 토큰이 필요합니다. 토큰은 이 브라우저에만 저장되고 저장소엔 들어가지 않습니다.
Fine-grained 토큰 생성 → github.com/settings/personal-access-tokens
• Repository access: scl660105-alt/newsnews 만 선택
• Permissions → Contents: Read and write (필수) · Actions: Read and write (선택 — "지금 적용" 버튼용)

2. 알림 키워드

이 단어 중 하나라도 제목·요약에 들어간 새 기사를 텔레그램 채널로 전송합니다.
고급 설정
OR (하나라도) AND (전부)

3. 저장 & 적용

저장하면 data/alerts.json이 갱신됩니다. 매시 자동수집 때 새 키워드가 적용되며, "지금 적용"을 누르면 수집·전송을 즉시 1회 실행합니다.
